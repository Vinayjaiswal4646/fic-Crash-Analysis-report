# 🚀 Traffic Crash Analysis Dashboard

## 📊 Overview
This project analyzes 200,000+ traffic crash records to identify patterns in:
- Driver behavior
- Vehicle impact
- Environmental conditions
- Road safety risks

## 🛠 Tools Used
- Power BI
- Data Cleaning & Feature Engineering
- Real-world dataset (Data.gov)

## 📈 Key Insights
- 48% crashes due to driver fault
- 30–40 mph zones are most risky
- Rain increases crash risk by 2–3x
- Rear-end & angle collisions = 70% cases
